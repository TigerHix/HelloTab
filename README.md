# HelloTab

A powerful and intuitive new tab extension for Chrome and Firefox.

![](https://i.imgur.com/TEOTayc.png)
![](https://i.imgur.com/kdQcvXJ.png)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
